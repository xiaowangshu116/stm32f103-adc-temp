## 一、MCU 完整引脚分配

### 时钟系统

| 引脚 | 网络名 | 功能 | 备注 |
|------|--------|------|------|
| 5 PD0 | OSC_IN | 外部晶振输入 | X1: 10MHz / 12pF / 30ppm |
| 6 PD1 | OSC_OUT | 外部晶振输出 | 串 390Ω → X1 |

### USART1 — 主 RS232 (接 J1)

| 引脚 | 网络名 | 复用功能 | 信号路径 |
|------|--------|---------|---------|
| PB6 (42) | TXD1_MCU | USART1_TX | R40 0Ω → U4 隔离 → U3 MAX3232 → J1 |
| PB7 (43) | RXD1_MCU | USART1_RX | J1 → U3 → U4 → R41 0Ω → MCU |

### USART2 — 调试 RS232 (接 J3, 标注 OPEN)

| 引脚 | 网络名 | 复用功能 | 备注 |
|------|--------|---------|------|
| PA2 (12) | TXD2_MCU | USART2_TX | R45 0Ω → U4 隔离 → U3 → J3 |
| PA3 (13) | RXD2_MCU | USART2_RX | J3 → U3 → U4 → R47 0Ω → MCU |

### ADC 采集 (5 路, 12位)

| 引脚 | 网络名 | ADC 通道 | 采集对象 |
|------|--------|---------|---------|
| PA0 (10) | — | ADC12_IN0 | 未使用 |
| PA1 (11) | — | ADC12_IN1 | 未使用 |
| PA4 (14) | **AD_SIC_I** | ADC12_IN4 | SIC 光源电流反馈 |
| PA5 (15) | **AD_SIC_V** | ADC12_IN5 | SIC 光源电压反馈 |
| PA6 (16) | **AD_TEMP_LIGHT** | ADC12_IN6 | PT100 温度采集 ⭐ |
| PA7 (17) | HEAT_EN | ADC12_IN7 | 加热器使能 (兼数字输出) |
| PB0 (18) | — | ADC12_IN8 | 未用 |
| PB1 (19) | SIC_EN | ADC12_IN9 | SIC 使能 (兼数字输出) |

### LED 指示灯 (共阳极, 3 路)

| 引脚 | 丝印 | 颜色 | 功能 | 电路 |
|------|------|------|------|------|
| PA8 (28) | LED1 | 🟢 绿 | RUN 运行指示 | R49 3kΩ 0.1% → +3.3VD |
| PB14 (27) | LED2 | 🔴 红 | ERR 错误指示 | R50 3kΩ 0.1% → +3.3VD |
| PB12 (25) | LED3 | 🟢 绿 | 状态指示 | R51 3kΩ 0.1% → +3.3VD |

> ⚠️ 共阳极: 低电平 (0) = 亮, 高电平 (1) = 灭。限流电阻 3kΩ, LED 电流约 1mA。

| PB5 (41) | DAC_SYN | SPI1_NSS | U17 SYNC |

### SIC 电机驱动 (SGM42500 H桥)

| PB10 (21) | VCM_BP | VCM B 相位置反馈 |
| PB11 (22) | VCM_AP | VCM A 相位置反馈 |

### SWD 编程调试口 (J6, 4P)

| 引脚 | 网络名 | 功能 |
|------|--------|------|

| PA14 (37) | SWCLK | 时钟线 (R46 10Ω) |

| 引脚 | 网络名 | 功能 |

## 三、温度采集回路

```
                        ┌─────────────────────────────────────┐
                        │           06-TEMP_CTL 模块           │
                        │                                     │
  REF_2.5V ────────────┤  U16 (SGM4029-2.5XS8G)              │
  精密基准源             │  ├─ 输出: 2.5V ±0.1%                   │
  (U18 SGM4029-4.096    │  └─ 供 U16 恒流源 + ADC 参考           │
   供 ADC VREF)          │                                     │
                        │  U16 恒流源                            │
                        │  ├─ 激励电流: 0.625mA                  │
                        │  ├─ PT100+ → J4 端子 PIN1              │
                        │  └─ PT100- → J4 端子 PIN2              │
                        │                                     │
        J4 ────────────┤  PT100 传感器 (外部)                    │
        PIN1: PT+       │  ├─ 0°C = 100Ω → V = 62.5mV           │
        PIN2: PT-       │  ├─ 300°C = 215.53Ω → V = 134.7mV     │
                        │  └─ 系数: 0.385Ω/°C                    │
                        │                                     │
                        │  信号调理 (运放)                        │
                        │  U7A (SGM8273-2XS8G)                   │
                        │  ├─ 输入: PT100 差分信号                 │
                        │  ├─ 增益: 17.47 (电路图标注)            │
                        │  └─ 输出 → U8 (SGM620) 仪表放大        │
                        │                                     │
                        │  U8 (SGM620XS8G/TR) 仪放              │
                        │  ├─ R28: 3k 0.1% 增益电阻              │
                        │  ├─ REF: 接参考电压                      │
                        │  └─ OUT → PA6 (AD_TEMP_LIGHT)         │
                        │                                     │
  PA6 ←───────────────┤  ADC 输入范围                           │
                        │  ├─ 0°C:   V_ADC ≈ 1.0919V            │
                        │  └─ 300°C: V_ADC ≈ 2.3532V            │
                        └─────────────────────────────────────┘

  温度计算公式:
    R_PT100 = 100 + 0.385 × T
    V_PT100 = 0.625mA × R_PT100 / 1000
    V_ADC = V_PT100 × 17.47
    ────────────────────────────
    T = ((V_ADC / 17.47 / 0.625 × 1000) - 100) / 0.385
```

### ADC 参考电压

| 信号 | 来源 | 芯片 | 精度 |
|------|------|------|------|
| REF_2.5V | 恒流源/运放偏置 | SGM4029-2.5XS8G | ±0.1% |
| REF_4.1V | ADC 参考 / DAC 参考 | SGM4029-4.096XS8G | ±0.1% |
| VREF_2.5V | PT100 电路 | — | — |

## 四、SIC 光源控制系统

### 恒流控制环路

```
                      ┌────────────────────────────────────────┐
                      │         05-LIGHT&HEAT_DRV 模块          │
                      │                                        │
  DAC_OUT ────────────┤  U17 (TPC112S1-VR) 12位 DAC             │
  (0~4.096V)          │  ├─ SPI 接口: SYNC/SCLK/DIN              │
                      │  └─ VOUT = 0 ~ 4.096V                   │
                      │                                        │
  SIC_EN ─────────────┤  光源使能控制                             │
  (PB1, 高有效)        │  ├─ Q3 (MMBT3906) PNP                   │
                      │  ├─ Q4 (MMBT3904) NPN                   │
                      │  └─ EN_SIC → Module7 降压模块使能         │
                      │                                        │
  HEAT_EN ────────────┤  加热器控制                               │
  (PA7, 高有效)        │  ├─ Module10 (CTRL-PMOS-BJT-4A)         │
                      │  └─ 输出: 24V / 最大 4A                  │
                      │                                        │
  SGM42500 ───────────┤  H桥 VCM 电机驱动                        │
  (U15)               │  ├─ PWM_A1, PWM_B1 → 电机线圈            │
                      │  ├─ Rs1=100mΩ 电流采样电阻               │
                      │  ├─ SIC+ / SIC- → J5 端子               │
                      │  └─ VCM_AP/BP → 位置反馈 ADC             │
                      │                                        │
  U14 (SGM620) ───────┤  电流检测仪放                             │
                      │  ├─ 增益: G = 1 + 49.4/R74 ≈ 17.5       │
                      │  ├─ 采样电压 ÷ 17.47 = SIC 电流          │
                      │  └─ OUT → PA4 (AD_SIC_I)                │
                      │                                        │
  功率估算:                                                     │
  ├─ SIC 光源: 24V × 1A ≈ 24W (最大 40W)                        │
  └─ 加热器:  24V × 4A ≈ 96W (最大)                              │
  └────────────────────────────────────────────────────────┘
```

### SIC 控制信号汇总

| 信号 | 方向 | 类型 | 说明 |
|------|------|------|------|
| DAC_SYN | MCU→DAC | SPI CS | TPC112S1 片选 |
| DAC_CLK | MCU→DAC | SPI SCK | TPC112S1 时钟 |
| DAC_DIN | MCU→DAC | SPI MOSI | TPC112S1 数据 |
| DAC_OUT | DAC→SGM42500 | 模拟 0-4.096V | 设定光源功率 |
| AD_SIC_I | SGM620→MCU | 模拟 | 光源电流 (PA4) |
| AD_SIC_V | 分压→MCU | 模拟 | 光源电压 (PA5) |
| SIC_EN | MCU→Q3/Q4 | 数字 | 光源使能 (PB1) |
| HEAT_EN | MCU→Module10 | 数字 | 加热使能 (PA7) |
| PWM_A1 | MCU→SGM42500 | PWM | VCM A 相 |
| PWM_B1 | MCU→SGM42500 | PWM | VCM B 相 |

## 八、GND 地线系统
